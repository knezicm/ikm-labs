# Laboratorijska vježba 1: Kroskompajliranje i debagovanje
Za realizaciju vježbe, neophodno je pokrenuti razvojnu platformu zasnovanu na *Ubuntu 16.04 LTS* operativnom sistemu, koja je preinstalirana i izvršava se kao virtuelna mašina pod nazivom *rpi-linux-dev* u okviru *VMWare Workstation Player* softvera za virtuelizaciju.

Nakon prijavljivanja na sistem, prvi korak je kloniranje udaljenog repozitorijuma u kojem se nalaze svi fajlovi neophodni za izradu laboratorijske vježbe. U tu svrhu, u terminalu je neophodno pokrenuti sljedeću sekvencu komandi:

```
git clone https://github.com/knezicm/ikm-labs/
```

Pretpostavlja se da se korisnik nalazi u `/home/student` direktorijumu.
